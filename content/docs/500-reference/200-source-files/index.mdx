此模块提供了使用本地文件作为内容源的方法。

典型用法包括三个主要功能:

- [`makeSource`](/docs/reference/source-files/make-source) 控制应用程序的整个模式定义。
- [`defineDocumentType`](/docs/reference/source-files/define-document-type) 定义架构顶级文档类型 (也称为*models*或*content types*).
- [`defineNestedType`](/docs/reference/source-files/define-nested-type) 定义可在多个文档类型定义中重用的数据形状的架构。
